Server migration, "feature" additions

06 Nov 2018 - Rick

siteupdate 
  • HTTP server migrated to lighttpd. This was a good idea no matter what considering I was using a python script as the prior http server. Setup was fairly easy and uptime is ~99.5% right now, loving it so far.
  • Loads of changes to the stylesheet since the site first rolled out
  • Implemented a system for tagging posts, still using nothing more than jekyll.
Might not seem like a lot, but considering I work on this website (( when I feel like it )), working 9 to 11 hours a day at my job, I feel like it's progress. As always, this whole website, including jekyll templates, are hosted on my Github.
