Windows

My least favorite operating system.

Why don't I like Windows? (and Microsoft in general)

  • If you buy a piece of hardware, it should be your property.

    • When you buy a computer from a computer retailer, what you are paying for is the hardware that makes up a computer system. It should be reasonable to say that you are allowed to run whatever software you desire on this hardware you purchased, and for most computers for most of the time that computers have been around, this was true.

  • Microsoft denies users partially or fully of this control

    • Around the time that Windows 8 came along, Microsoft started getting manufacturers to ship computers running Windows with a UEFI feature called Secure Boot enabled by default. Secure Boot was intended to be a security feature which effectively prevented any other operating system from being installed onto the computer by blocking "unverified software," meaning that in order to be ran on this hardware, a specific linux distribution must beg microsoft to let their distribution boot. Some manufacturers even made it either very difficult or plain impossible to disable Secure Boot with the version of UEFI they used.

  • On Microsoft's Windows 7, 8 and 10 operating systems, collection of data from users is rampant.

    • Microsoft slurps up user data like an excited kid drinking a milkshake. Almost anything you do while in a Windows environment is collected and sent to the mothership, and if you don't like it, too bad, because there's a good chance it's either really complicated or impossible to change operating systems now.

  • On Windows 10, Microsoft just slaps ads right there in your menus.

    • As if Windows didn't have enough problems, Microsoft decided to go ahead and throw in some ads to pollute your screen and make them money. You could argue this is somewhat okay because Windows 10 is free, but what about the people who payed for an older version and were forced to upgrade to Windows 10 anyways?
